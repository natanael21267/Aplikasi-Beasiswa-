Scholarship Decision Support System

A desktop-based Scholarship Decision Support System developed using Java NetBeans and the SMART (Simple Multi Attribute Rating Technique) method. This application helps automate the scholarship selection process by evaluating applicants based on multiple criteria and generating ranked results to support objective decision-making.

Features
Scholarship applicant data management
Criteria and weight management
SMART method calculation process
Automatic applicant ranking
Decision result visualization
Desktop-based user interface
Technologies Used
Java
Java Swing
NetBeans IDE
MySQL (if used)
SMART Method
System Purpose

This application was created to simplify the scholarship selection process by reducing manual calculations and improving accuracy in determining eligible scholarship recipients.

Criteria Example
Academic grades
Parents’ income
Attendance
Achievements
Discipline
Output

The system generates a ranking list of scholarship candidates based on the final SMART calculation results.
